збекистан AI Excellence Center Platform

Современный прототип образовательной экосистемы, объединяющий преподавателей и студентов Узбекистана для обучения и исследований в области искусственного интеллекта.

Основные возможности

Проект реализован как SPA (Single Page Application), что обеспечивает мгновенное переключение между ролями без перезагрузки страницы.

Landing Page (Главная)
— Светлая тема: Использование Glassmorphism и мягких градиентов для современного вида.
— Статистика: Интерактивные блоки с ключевыми показателями платформы.
— Адаптивность: Полная поддержка мобильных устройств благодаря Tailwind CSS.

Панель Преподавателя (Dashboard)
— Загрузка контента: Поддержка Drag-and-Drop для видео, статей и датасетов.
— AI Интеграция: Функции автоматической генерации описаний и субтитров.
— Аналитика: Мониторинг просмотров и управление статусами материалов.

Панель Студента (Dashboard)
— Библиотека курсов: Персонализированные карточки обучения (Prompt Engineering, NLP, Computer Vision).
— Трекинг прогресса: Визуальные индикаторы завершения уроков и получения сертификатов.
— Поиск: Интеллектуальный фильтр по категориям и уровню сложности.

Технологический стек

— Frontend: HTML5, Vanilla JavaScript.
— Styling: Tailwind CSS (с кастомными конфигурациями градиентов).
— UI/UX: Glassmorphism Design, шрифты Inter, SVG иконки.
— SDK: Интегрирован Playables SDK для отслеживания жизненного цикла приложения.
